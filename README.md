# Calculator App

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. This project was created to strengthen frontend development skills by practicing DOM manipulation, event handling, CSS Grid, Flexbox, and JavaScript logic.

## 🚀 Live Demo

https://calculator-app-vwnh.onrender.com

## ✨ Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Percentage (%) calculations
* Automatic bracket handling
* Clear (AC) functionality
* Backspace support
* Formatted output with comma separators
* Responsive user interface
* Modern dark theme design

## 🛠️ Technologies Used

## ⚙️ How It Works

1. User clicks a calculator button.
2. JavaScript reads the button value using `data-key`.
3. Input is validated before being added.
4. Operators and symbols are formatted for display.
5. Expression is evaluated when `=` is pressed.
6. Output is formatted and displayed to the user.
